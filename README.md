# YaDOM Website

Product landing page for YaDOM Herbal Inhaler (ยาดมสมุนไพร).

## Features

- Hero banner with gradient backgrounds
- Product showcase with 4 products
- Customer reviews/testimonials
- Newsletter signup
- Responsive design

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
